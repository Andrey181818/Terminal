Linux terminal (GitBash) commands. Terminal HW_1

1) Посмотреть где я
   $ pwd

2) Создать папку
   $ mkdir terminal

3) Зайти в папку
   $ cd terminal

4) Создать 3 папки
   $ mkdir dir_1 dir_2 dir_3

5) Зайти в любую папку
   $ cd dir_1

6) Создать 5 файлов (3 txt, 2 json)
   $ touch 01.txt 02.txt 03.txt 04.json 05.json

7) Создать 3 папки
   $ mkdir kot_1 kot_2 kot_3

8) Вывести список содержимого папки
   $ ls -la

9) + Открыть любой txt файл
   $ cat > 01.txt

10) + Написать туда что-нибудь, любой текст
   $ Andrei Lora

11) + Сохранить и выйти
   $ Enter, Ctrl + C

12) Выйти из папки на уровень выше
   $ cd ..

	Выйти из папки на уровень ниже(вглубь)
        $ cd dir_1

13) Переместить любые 2 файла, которые вы создали, в любую другую папку
   $ mv 04.json 05.json ./kot_1 (в папку вглубь)    или    mv 04.json 05.json ../ (в папку выше)

14) Скопировать любые 2 файла, которые вы создали, в любую другую папку
   $ cp 01.txt 02.txt ./kot_2 (в папку вглубь)    или    cp 01.txt 02.txt ../ (в папку выше)

15) Найти файл по имени
   $ find -name 05.json (выводит путь относительно текущей директории)

16) Просмотреть содержимое в реальном времени
   $ tail -f 01.txt

17) Вывести несколько первых строк из текстового файла
   $ head -3 01.txt (выведет первые 3 строки)

18) Вывести несколько последних строк из текстового файла
   $ tail -3 01.txt (выведет последние 3 строки)

19) Просмотреть содержимое длинного файла
   $ less 01.txt (Выход :q)

20) Вывести дату и время
   $ date

Linux terminal (GitBash) commands. Terminal HW_2

1. Сделать папку dir_1
   $ mkdir dir_1

2. Зайти в папку dir_1
   $ cd dir_1

3. Создать папку inner_dir_1
   $ mkdir inner_dir_1

4. Посмотреть где ты находишься
   $ pwd

5. Находясь в папке dir_1 создать пустой текстовый файл tf_1.txt
   $ touch tf_1.txt

6. Находясь в папке dir_1 через команду cat создать текстовый файл tf_2.txt со следующими строками:
- the first 1 - the second 2 - the third 3
   $ cat > tf_2.txt
   the first 1
   the second 2
   the third 3
   Ctrl + C

7. Зайти в папку inner_dir_1
   $ cd inner_dir_1

8. Через cat сделать текстовый файл tf_3.txt c любыми строками
   $ cat > tf_3.txt
   ...
   Ctrl + C

9. Через cat добавить в текстовый файл tf_3.txt строку “the second 2”
   $ cat >> tf_3.txt
   the second 2
   Ctrl + C

10. Через cat добавить в текстовый файл tf_3.txt строку “the sec 2”
   $ cat >> tf_3.txt
   the sec 2
   Ctrl + C

11. Через cat добавить в текстовый файл tf_2.txt строку “the sec 3”
   $ cat >> tf_2.txt
   the sec 3
   Ctrl + C

12. Через cat добавить в текстовый файл tf_3.txt строку “the SeCoNd 2”
   $ cat >> tf_3.txt
   the SeCoNd 2
   Ctrl + C

13. Через cat добавить в текстовый файл tf_2.txt строку “the seConD 2”
   $ cat >> tf_2.txt
   the seConD 2
   Ctrl + C

14. Сделать текстовый файл tf_4.txt в котором будет 15 строк
   $ for i in {1..15}; do echo "This is line $i" >> tf_4.txt; done

15. Сделать текстовый файл tF_5.txt в котором будет 13 строк
   $ for i in {1..13}; do echo "This is line $i" >> tF_5.txt; done

16. Вывести список всех файлов в папке
   $ ls -la

17. Выйти из папки inner_dir_1
   $ cd ..

18. Вывести содержимое файла tf_3.txt в терминал
   $ cat ./inner_dir_1/tf_3.txt

19. Найти путь к файлу tf_4.txt
   $ find -name tf_4.txt

20. Очистить файл tf_4.txt от содержимого без удаления самого файла
   $ cat > tf_4.txt
   Ctrl + C

21. Найти путь к файлам у которых есть “tf” в названии
   $ find -name '*tf*'

22. Найти путь к файлам у которых есть “tf” в названии и буквы в любом регистре
   $ find -iname '*tf*'

25. Найти строки в файлах где есть только комбинация букв “sec” в текущей папке
